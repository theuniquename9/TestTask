# **TCP-сервис для передачи данных**

---

## **Описание проекта**

```plaintext
Этот проект реализует TCP-сервис для передачи и приема данных. Основная цель — создание эффективного и удобного инструмента для работы с сетевыми соединениями. Сервис поддерживает два режима работы:
1. **Отправитель** — устанавливает соединение и отправляет данные.
2. **Слушатель** — принимает данные, декодирует их и выводит ключевую информацию, включая размер и первые 16 байт переданного блока.

Функционал разделен на два класса:
- `MySender` — для передачи данных через TCP-соединение.
- `MyListener` — для приема данных, обработки информации о размере и выводе первых байт данных.

## **Структура проекта**

src/
├── main.py                   # Точка входа. Обрабатывает аргументы командной строки и запускает отправитель или слушатель.
├── testtask/
│   ├── my_sender.py          # Класс MySender для TCP-соединения и передачи данных.
│   ├── my_listener.py        # Класс MyListener для приема данных через TCP.
run_test.sh                   # Bash-скрипт для тестирования одного слушателя и отправителя.

Класс `MySender` отвечает за установление TCP-соединения и передачу данных. Основные особенности:
- **Динамическое кодирование размера данных:** Использует минимально возможное количество байт для передачи размера данных, снижая нагрузку.
- **Настраиваемый размер данных:** По умолчанию ограничение составляет 1 КБ для тестирования, но оно может быть изменено в константе `MAX_SIZE`.


Класс `MyListener` реализует TCP-сервер, который:
- Принимает данные от отправителя.
- Декодирует длину данных, используя переменное количество байт.
- Выводит размер данных и первые 16 байт для проверки.


`run_test.sh` автоматизирует запуск одного слушателя и одного отправителя для локального тестирования. По умолчанию использует `127.0.0.1` (localhost) для простоты.

Использование
Запуск отправителя
bash
Копировать код
python src/main.py sender <server_address> <port>
Запуск слушателя
bash
Копировать код
python src/main.py listener <ip_address> <port>
Пример
plaintext
Копировать код
1. Запустите слушателя:
   python src/main.py listener 127.0.0.1 12345

2. Запустите отправителя:
   python src/main.py sender 127.0.0.1 12345
Запуск автоматического тестирования
bash
